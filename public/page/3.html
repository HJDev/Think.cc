Ubuntu 16.0.4 安装 Swift 后提示 error while loading shared libraries: libpython2.7.so.1.0

Ubuntu 16.0.4 安装 Swift 后提示: error while loading shared libraries: libpython2.7.so.1.0: swift/usr/bin/lldb: error while loading shared libraries: libpython2.7.so.1.0: cannot open shared object file: No such file or directory 这个问题会在 Ubuntu 14.04 和 Ubuntu 16.04 上出现，是 swift 的一个依赖问题，只需要安装 libpython2.7-dev 就可以解决问题。代码如下 sudo apt-get install libpython2.7-dev Done !
